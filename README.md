# interview-concepts

# Java: 
Init array: 
```
int array[] = { 1, 2, 3, 4, 5 };
```
Note: It's not possible to initialize an array after the declaration using this approach. An attempt to do so will result in a compilation error.

Init 2d array:
```
String[][] names = { 
                    {"John", "Smith"}, 
                    {"Javin", "Paul"}, 
                    {"James", "Gosling"}, 
                   };
```
